How to burn image for OlimexIoT server

Back to How to install OlimexIoT server

Download image

OlimexIoT Server - A20-OLinuXino-MICRO Image

SD Card requirements

  • Class 10

Linux

NOTE You MUST replace /dev/sdX with your card reader device, e.g. /dev/sdb OR /dev/sdc

Plug your SD-card into your computer

To determine your card reader device use following command

lsblk -d -o NAME,SIZE,TRAN,LABEL

Unmount partitions if there are any automounted

sudo umount /dev/sdX*

To burn image use following command

sudo dd bs=4M oflag=sync if=OlimexIoT.img of=/dev/sdX

Windows

To write a Linux image to an SD card under Windows you can use Win32 Disk Imager.

  1. Download Win32 Disk Imager and install it.
  2. Run the program
  3. Select image file you have downloaded and SD card device.
  4. Click on Write button.
