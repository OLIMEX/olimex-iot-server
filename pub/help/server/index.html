Back to OlimexIoT help

In this article we will describe how to build your own OlimexIoT server.

Hardware used

  • A20-OLinuXino-MICRO
  • MOD-WIFI-R5370
  • MICRO-SD-8GB-CLASS10
  • SATA-HDD-2.5-500GB
  • SATA-CABLE-SET
  • USB-MINI-CABLE
  • Power supply adapter 12V/0.5A
  • BATTERY-LIPO1400mAh
  • BOX-MICRO-B-BLACK

Installation options

You can choose between various options

  • Use our pre-build SD card
  • Download image and burn it yourself
  • Build the u-boot, kernel and rootfs yourself (advanced users)
