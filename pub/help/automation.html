Automation

Back to OlimexIoT Help

IoT Node connected to the OlimexIoT Server sends a message to the server for each event that occurred. These messages are used to monitor and store the state of IoT Devices. On the other hand some events may occur on the server itself.

This approach gives OlimexIoT Server the ability to achieve certain level of automation trough two main features.

  • Actions
  • Triggers

Triggers can be categorised in two groups:

  • Event driven
  • Time scheduled

Event driven

If temperature is bellow 20°C turn the heater on.
If temperature is over 25°C switch the heater off.
If measured voltage is over 250V switch the relay off to protect my TV.
If connection to a node is lost send me an e-mail message.

Time scheduled

Every minute display the temperature on MOD-LED8x8RGB.
Every day at 19:00 switch on lights.
